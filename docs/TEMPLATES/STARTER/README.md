## Want More Advanced Templates?

A separate [Builder Pack](https://ashpdx.gumroad.com/l/modular-markup) also exists with more advanced offline browser apps, including:
- Kanban boards
- MBOX email viewer
- Offline GPT/chat archive viewer
- Quiz builder
- Quote image generator
- Topic clustering tools
- and more

The free pack is intentionally fully usable on its own.

If you enjoy the ecosystem and want additional tools, you can explore the Builder Pack here:
[https://ashpdx.gumroad.com/l/modular-markup](https://ashpdx.gumroad.com/l/modular-markup)
