Chicago Citation Style Guide

Notes and Bibliography system

Source Type Bibliography Format Footnote / Endnote Format Key Chicago Conventions
Book Last Name, First Name. Title of Book. Publisher, Year. 1. First Name Last Name, Title of Book (Publisher, Year), 23. Bibliography uses inverted author name.
Notes use normal name order.
Specific page numbers usually go in notes.
Journal Article Last Name, First Name. "Title of Article." Journal Title volume, no. issue (Year): xx-xx. DOI or URL. 1. First Name Last Name, "Title of Article," Journal Title volume, no. issue (Year): 23, DOI or URL. Article title in quotation marks.
Journal title italicized.
Notes cite the specific page referenced; bibliography gives the full page range.
Webpage Last Name, First Name. "Title of Page." Website Name. Accessed Month Day, Year. URL. 1. First Name Last Name, "Title of Page," Website Name, accessed Month Day, Year, URL. Include publication or revision date when available.
Access date is often used for webpages.
URL goes at the end.
Chapter in Edited Book Last Name, First Name. "Title of Chapter." In Title of Book, edited by First Name Last Name, xx-xx. Publisher, Year. 1. First Name Last Name, "Title of Chapter," in Title of Book, ed. First Name Last Name (Publisher, Year), 23. Chapter title in quotation marks.
Book title italicized.
Notes cite the page used; bibliography includes full chapter page range.
Two or Three Authors Last Name, First Name, and First Name Last Name. Title. Publisher, Year. 1. First Name Last Name and First Name Last Name, Title (Publisher, Year), 23. List all authors in both bibliography and notes for short author lists.
Four or More Authors Last Name, First Name, First Name Last Name, First Name Last Name, and First Name Last Name. Title. Publisher, Year. 1. First Name Last Name et al., Title (Publisher, Year), 23. Notes often shorten longer author lists with et al..
Bibliography usually lists the full author list unless using a shortened house style.
Direct Quote Use the normal bibliography entry for the source. 1. First Name Last Name, Title (Publisher, Year), 23. Chicago Notes-Bibliography uses note numbers in the text.
The note gives the exact page cited.
No Author Title of Work. Publisher, Year. 1. Title of Work (Publisher, Year), 23. Move the title to the front when no author is listed.
No Date Last Name, First Name. "Title of Page." Website Name. Accessed Month Day, Year. URL. 1. First Name Last Name, "Title of Page," Website Name, accessed Month Day, Year, URL. For undated web sources, Chicago commonly relies on an access date.
Repeated Citation Source appears once in the bibliography. 2. Last Name, Shortened Title, 45. Later notes are often shortened rather than repeated in full.
General Chicago Pattern Author. Title. Publisher, Year. 1. Author, Title (Publisher, Year), page. Notes and bibliography entries are related, but not identical.
Bibliography entries end with a period; notes often separate elements with commas.
