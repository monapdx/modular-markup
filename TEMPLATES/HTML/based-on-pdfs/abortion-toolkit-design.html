Who this style is for

Great for toolkits, landing pages, guides, activism resources, app explainers, and GitHub project pages that need warmth and structure.

What makes it work

Rounded panels. Big type. Layered depth. Limited palette. Tinted imagery. Repetition without feeling stiff.

Core rule

Build every section like a stack of friendly UI cards instead of a flat document.

Layout rules

Use cards, not boxes.

Think in layers. A large base card, a second offset card, then a highlight card. Keep corners generous and spacing roomy.

  • Radius: 20–30px for nearly everything
  • Spacing: 24px minimum inside content blocks
  • Hierarchy: 1 giant headline, 1 section title, 1 body style
  • Palette: one dark anchor, one cream neutral, 2–4 accent colors
  • Images: always tint or darken them for cohesion
