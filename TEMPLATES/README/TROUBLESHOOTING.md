# Troubleshooting

## Common Problems

### Problem: Installation fails
**Possible causes**
- Missing dependency
- Wrong runtime version
- Broken lockfile

**Try this**
```bash
<example-fix-command>
```

### Problem: App will not start
**Possible causes**
- Missing environment variables
- Port conflict
- Build output missing

**Try this**
- Check your config
- Review terminal errors
- Reinstall dependencies

### Problem: Feature behaves incorrectly
**Try this**
- Refresh / restart
- Clear cache
- Confirm reproduction steps
- Check known issues

## Debug Checklist
- Confirm required software versions
- Confirm environment variables are set
- Confirm install completed successfully
- Check logs / browser console / terminal output

## Still Stuck?
Open an issue or use the support channel listed in `SUPPORT.md`.
