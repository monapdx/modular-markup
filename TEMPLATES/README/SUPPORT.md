## Need Help?
Explain how users can get help with this project.

## Before Reaching Out
Please check:
- `README.md`
- `FAQ.md`
- `TROUBLESHOOTING.md`
- Existing issues / discussions

## Support Channels
- GitHub Issues:
- GitHub Discussions:
- Email:
- Community chat / Discord / forum:

## What to Include
When asking for help, include:
- What you were trying to do
- What happened instead
- Error messages
- Screenshots if relevant
- Your operating system / browser / runtime version

## Response Expectations
Set expectations around response times and the types of support you do or do not provide.

## Security Issues
For security vulnerabilities, do **not** open a public issue. See `SECURITY.md`.
