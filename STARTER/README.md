# Starter Pack

Local-first HTML browser tools you can open, edit, and save offline. Changes persist in your browser; most templates include **Export This Version** and **Restore Default**.

⭐ [Star on GitHub](https://github.com/monapdx/modular-markup) · [Browse templates](index.html)

## Enjoying the Starter Pack?

You are using the free [Starter Pack](https://github.com/monapdx/modular-markup).

## Want more advanced templates?

The separate **[Builder Pack](https://ashpdx.gumroad.com/l/modular-markup)** adds offline apps and generators on top of the free pack. The Starter Pack is fully usable on its own.

### Builder root tools

| File | What it does |
|------|----------------|
| `calendar.html` | Editable month calendar with one event per day |
| `kanban.html` | Drag-and-drop Kanban board with search and priorities |
| `worksheet.html` | Printable neo-brutalist worksheet and workshop tracker |
| `masonry-gallery.html` | Import images into a masonry gallery; export standalone HTML |
| `mindmap.html` | Pan/zoom mind map with multiple nodes and SVG connectors |
| `offline-GPT.html` | Browse exported AI chat JSON archives offline |
| `newsletter-builder.html` | Table-based email newsletter layouts for Gmail paste |

### Builder folders

| Path | Contents |
|------|----------|
| `activity-tracker/` | Premium copy of the daily activity tracker |
| `dashboard/` | Dashboard layout shells plus the **widgets** collection |
| `dashboard/widgets/` | Generator tools (FAQ, CSV→SQL, fake data, regex explainer, quiz builder, quote images, and more) |
| `sticker-sheet/` | GIF/PNG sticker grid importers with export and restore |
| `mbox-viewer/` | **Premium** multi-file MBOX viewer (HTML + CSS + JS) — larger archives, streaming index |
