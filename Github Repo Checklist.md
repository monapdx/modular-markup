A good GitHub repo isn’t just code—it explains itself and invites others to use or contribute.

- [ ] **Must Have (start here):**
  - [ ] **Description** (350 characters)
  - [ ] **Topics** (20 max)
  - [ ] **README.md**
	  - [ ] What it does?
	  - [ ] How to run it?
	  - [ ] Screenshots/Demo
  - [ ] LICENSE
  - [ ] .gitignore

- [ ] **Should Have (for collaboration):**
  - [ ] CODE_OF_CONDUCT.md
  - [ ] CONTRIBUTING.md
  - [ ] SECURITY.md


- [ ] **Nice to Have:**
  - [ ] A few starter **Issues** & **Discussions**
  - [ ] Live demo on **GitHub Pages**
  - [ ] Good **commit messages**
  - [ ] **Social preview image**
	  - [ ] 1280x640 pixels
	  - [ ] Under 1MB
